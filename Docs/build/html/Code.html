class PyMieSim.Mesh.FibonacciMesh(MaxAngle: float = 0.5235987755982988, Sampling: int = 1000, PhiOffset: float = 0, GammaOffset: float = 0)[source]¶

Class wich represent an angular mesh. The distribution of points inside the mesh is similar to a Fibonacci sphere where each point cover an equivalent solid angle.

MaxAngle : float
Angle in radian defined by the numerical aperture of the imaging system.
Sampling : int
Number of point distrubuted inside the Solid angle defined by the numerical aperture.
PhiOffset : float
Angle offset in the parallel direction of the polarization of incindent light.
GammaOffset : float
Angle offset in the perpendicular direction of the polarization of incindent light.
