GetEfficiencies()¶

Methode compute all Efficiences (\(Q_{sca}, Q_{ext}, Q_{abs}\)) for the scatterer.

Qabs¶

Absorption efficiency: \(Q_{abs}=Q_{ext}-Q_{sca}\)

Qext¶

Extinction efficiency: \(Q_{ext}=\frac{2}{x^2}\sum_{n=1}^{n_{max}}(2n+1) / \text{real} \{ a_n+b_n \}\)

Qsca¶

Scattering efficiency: \(Q_{sca}=\frac{2}{x^2}\sum_{n=1}^{n_{max}}(2n+1)(|a_n|^2+|b_n|^2)\)

S1S2(Num=200)¶

Method compute \(S_1(\phi)\) & \(S_2(\phi)\). For spherical Scatterer such as here S1 and S2 are computed as follow:

\(S_1=\sum\limits_{n=1}^{n_{max}} \frac{2n+1}{n(n+1)}(a_n\pi_n+b_n\tau_n)\)

\(S_2=\sum\limits_{n=1}^{n_{max}}\frac{2n+1}{n(n+1)}(a_n\tau_n+b_n\pi_n)\)

Parameters:
Num : int

Number of point (\(\phi\)) to evaluate \(S_1\) & \(S_2\).
Returns:

an(MaxOrder=5)[source]¶

Compute \(a_n\) coefficient as defined in Eq:III.88 of B&B:

\(a_n = \frac{ \mu_{sp} \Psi_n(\alpha) \Psi_n^\prime(\beta) - \mu M \Psi_n^\prime(\alpha) \Psi_n(\beta)} {\mu_{sp} \xi_n(\alpha) \Psi_n^\prime(\beta)- \mu M \xi_n^\prime (\alpha) \Psi_n(\beta)}\)

With \(M = \frac{k_{sp}}{k}\) (Eq:I.103)

With \(M = \frac{k_{sp}}{k}\) (Eq:I.103)

Anm(n, m, Sampling=200)[source]¶

From ref[2]:Eq:18-19 Method returns:

\(A_{mn} = \gamma_0 \int_0^{\pi} \gamma_1 \big[ \hat{I}_{m+1} (\beta) + \hat{I}_{m-1} (\beta) \big] -\gamma_3 \hat{I}_{m} (\beta) d \theta\)

Where:

\(\gamma_0 = \frac{(-i)^n \rho_n^2}{(2n+1) \psi_n(\rho_n)}e^{-i Z_0}\)

\(\gamma_1 = Q \exp \big[ i Q s^2 \Big( R_0 - \rho_n \sin (\theta ) \big)^2 + i \rho_n \cos( \theta ) \Big] \hat{P}_n^{|m|} (\cos ( \theta ) ) \sin(\theta)\)

\(\gamma_2 = \big( 2 Q s^2 \rho_n \cos(\theta) -1 \big) \sin(\theta)\)

\(\gamma_3 = 4 Q s^2 X_0 \cos(\theta)\)

\(\gamma_4 = 4 Q s^2 Y_0 \cos(\theta)\)

\(I_m(\beta) = \frac{1}{2\pi} \int_0^{2\pi} e^{z \cos(\phi) - i m \phi} d\phi\)

\(\tilde{I}_m(\beta) = e^{-\beta - i m \xi} I_m(\beta)\)

\(Q_{spherical} = \frac{1}{2 ( r \cos (\theta) - z_0 )/k w_0^2 - i}\)
